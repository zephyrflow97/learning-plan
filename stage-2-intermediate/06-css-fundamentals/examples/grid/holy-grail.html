💡 关键代码:
grid-template-areas 让布局一目了然！

主内容区

这是圣杯布局的主内容区。它占据了页面的大部分空间。

圣杯布局的特点：

  • 头部和底部占据整行
  • 左侧导航固定宽度（200px）
  • 右侧边栏固定宽度（200px）
  • 中间主内容自适应剩余空间
  • 响应式友好（移动端变为单列）
Grid 实现:
display: grid;
grid-template-columns: 200px 1fr 200px;
grid-template-rows: auto 1fr auto;
grid-template-areas:
  "header header header"
  "nav    main   aside"
  "footer footer footer";

对比传统方案:
在 Float 时代（2005-2012），实现圣杯布局需要负 margin、clearfix hack、精确的宽度计算等技巧，代码量超过 50 行，Bug 无数。Grid 让这一切变得简单。
