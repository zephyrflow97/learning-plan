响应式导航栏示例

这个导航栏使用 Flexbox 实现了响应式布局：

  • 桌面端（>768px）: Logo 左对齐，菜单右对齐，水平排列
  • 移动端（≤768px）: 垂直堆叠，菜单项居中

关键代码:

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

@media (max-width: 768px) {
  .navbar {
    flex-direction: column;
  }
}
