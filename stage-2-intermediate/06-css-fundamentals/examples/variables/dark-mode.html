暗黑模式示例

点击右上角按钮切换主题

功能特性

  • 使用 CSS 变量实现主题切换
  • 状态持久化到 localStorage
  • 平滑的过渡动画
  • 完整的颜色系统

技术实现

核心思想：定义两套 CSS 变量，通过切换根元素的 class 来切换主题。

/* 亮色主题 */ :root { --bg: #ffffff; --text: #111827; --card-bg: #f9fafb; } /* 暗色主题 */ :root.dark { --bg: #111827; --text: #f9fafb; --card-bg: #1f2937; } /* 使用变量 */ body { background: var(--bg); color: var(--text); }

为什么使用 CSS 变量？

相比 Sass 变量，CSS 变量的优势：

  • 可以在运行时动态修改（通过 JavaScript）
  • 支持继承和作用域
  • 在媒体查询中可以修改值
  • 在浏览器 DevTools 中可见和调试
