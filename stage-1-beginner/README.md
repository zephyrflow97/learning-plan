# 阶段 1：入门级 - JavaScript 基础

欢迎来到 JavaScript 学习的第一阶段!在这个阶段,你将掌握 JavaScript 的核心语法和基本概念,为后续的学习打下坚实的基础。

## 🎯 学习目标

完成本阶段后,你将能够:

- ✅ 掌握 JavaScript 基础语法和数据类型
- ✅ 理解函数、作用域和闭包
- ✅ 熟练使用数组和对象操作
- ✅ 理解异步编程基础(Promise 和 async/await)
- ✅ 能够进行基本的 DOM 操作和事件处理
- ✅ 独立完成一个待办事项应用项目

## 📚 学习内容

### [第 1 章：JavaScript 基础语法](./01-javascript-basics/)

**学习时长：** 2-3 天

**核心内容：**
- 变量声明(var, let, const)
- 数据类型(原始类型和引用类型)
- 运算符和表达式
- 控制流(if/else, switch, 循环)

**学习成果：**
- 理解 JavaScript 的基本语法规则
- 能够正确使用变量和数据类型
- 掌握控制流程的方式

---

### [第 2 章：函数和作用域](./02-functions-scope/)

**学习时长：** 3-4 天

**核心内容：**
- 函数声明和表达式
- 箭头函数
- 作用域和闭包
- this 关键字和绑定

**学习成果：**
- 理解多种函数定义方式
- 掌握作用域链和闭包的原理
- 能够正确处理 this 绑定

---

### [第 3 章：对象和数组操作](./03-objects-arrays/)

**学习时长：** 3-4 天

**核心内容：**
- 对象字面量和属性访问
- 数组方法(map, filter, reduce 等)
- 解构赋值
- 展开运算符

**学习成果：**
- 熟练操作对象和数组
- 掌握常用数组方法
- 理解解构和展开运算符的使用

---

### [第 4 章：异步编程入门](./04-async-basics/)

**学习时长：** 3-4 天

**核心内容：**
- 回调函数
- Promise 基础
- async/await 语法
- 错误处理

**学习成果：**
- 理解异步编程的概念
- 掌握 Promise 和 async/await
- 能够处理异步操作的错误

---

### [第 5 章：DOM 操作基础](./05-dom-basics/)

**学习时长：** 2-3 天

**核心内容：**
- DOM 选择器和元素操作
- 事件监听和处理
- 动态创建和修改元素
- localStorage 存储

**学习成果：**
- 能够操作 DOM 元素
- 掌握事件处理机制
- 理解浏览器存储 API

---

### [第 6 章：原型链与原型继承](./06-prototype-inheritance/)

**学习时长：** 3-4 天

**核心内容：**
- JavaScript 对象的 `[[Prototype]]` 机制
- 原型链查找机制
- 构造函数与 `new` 关键字的四步骤
- `prototype` vs `__proto__` vs `Object.getPrototypeOf()`
- 原型链 vs ES6 Class 的本质区别
- 原型式继承与 `Object.create()`
- `instanceof` 的原型链遍历原理

**学习成果：**
- 理解 JavaScript 对象系统的底层机制
- 掌握原型链查找规则
- 能够解释 `new` 关键字的工作原理
- 理解 ES6 Class 是原型的语法糖
- 能够不用 Class 关键字实现继承

**为什么重要：**
- 原型链是 JavaScript 对象系统的核心
- 理解原型是掌握 JavaScript 高级特性的基础
- ES6 Class、装饰器等特性都基于原型机制

---

### [实战项目：待办事项应用](./projects/todo-app/)

**项目时长：** 3-5 天

**项目目标：**
创建一个功能完整的待办事项应用,支持:
- ➕ 添加新任务
- ✏️ 编辑任务内容
- ✅ 标记任务完成/未完成
- 🗑️ 删除任务
- 🔍 过滤任务(全部/进行中/已完成)
- 💾 数据持久化(localStorage)

**涵盖知识点：**
- DOM 操作和事件处理
- 数组方法(filter, map)
- localStorage API
- 函数组织和模块化
- 错误处理和数据验证

## 🎓 学习建议

### 学习顺序

建议按照章节顺序学习,不要跳过:

1. **第 1-3 章** 是基础,必须掌握
2. **第 4 章** 异步编程比较重要,需要多练习
3. **第 5 章** DOM 操作为项目做准备
4. **项目** 综合运用所有知识点
5. **练习题** 巩固和检验学习成果

### 学习方法

每个章节的学习流程:

1. **阅读讲解** - 理解概念和原理
2. **运行示例** - 在本地运行所有代码示例
3. **动手实践** - 修改示例代码,观察结果
4. **完成练习** - 独立完成章节练习
5. **总结笔记** - 用自己的话总结知识点

### 时间安排

**总学习时长：** 2-3 周

- **每天学习时间：** 2-3 小时
- **理论学习：** 40%
- **实践编码：** 60%

**每周安排建议：**
- 第 1 周：第 1-3 章
- 第 2 周：第 4-5 章
- 第 3 周：第 6 章 + 开始项目
- 第 4 周：完成项目 + 练习题

### 常见问题

**Q: 如果某个概念理解不了怎么办?**  
A: 不要着急,可以:
1. 重新阅读讲解,特别是基础解释部分
2. 运行代码示例,观察输出结果
3. 查看常见问题解答
4. 在实践中慢慢理解

**Q: 需要记住所有的 API 吗?**  
A: 不需要!重点是理解概念和原理,常用 API 在实践中会自然记住,不常用的查文档即可。

**Q: 代码示例运行出错怎么办?**  
A: 参考[调试和问题排查指南](../teaching-support/debugging-guide.md),学习如何:
1. 阅读错误信息
2. 使用 console.log 调试
3. 使用浏览器开发者工具
