## 📚 推荐书籍

### JavaScript
1. **《JavaScript 高级程序设计》（第 4 版）**
   - 作者：Matt Frisbie
   - 别名：红宝书
   - 适合：初学者到中级

2. **《你不知道的 JavaScript》系列**
   - 作者：Kyle Simpson
   - 深入讲解 JavaScript 核心机制
   - 适合：中级到高级

3. **《JavaScript 权威指南》（第 7 版）**
   - 作者：David Flanagan
   - 别名：犀牛书
   - 适合：作为参考书

4. **《Eloquent JavaScript》**
   - 作者：Marijn Haverbeke
   - 免费在线阅读
   - 适合：初学者

### TypeScript
1. **《TypeScript 编程》**
   - 作者：Boris Cherny
   - 全面讲解 TypeScript
   - 适合：中级

2. **《深入理解 TypeScript》**
   - 作者：Basarat Ali Syed
   - 免费在线阅读
   - 适合：中级到高级

### 设计模式和架构
1. **《JavaScript 设计模式与开发实践》**
   - 作者：曾探
   - 讲解 JavaScript 中的设计模式

2. **《重构：改善既有代码的设计》**
   - 作者：Martin Fowler
   - 代码重构经典

3. **《代码整洁之道》**
   - 作者：Robert C. Martin
   - 编写高质量代码
