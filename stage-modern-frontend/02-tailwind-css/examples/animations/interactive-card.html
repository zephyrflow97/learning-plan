交互效果详解

🖼️ 图片效果

  • overflow-hidden - 裁剪溢出内容
  • group-hover:scale-110 - 悬停时放大 110%
  • transition-transform duration-300 - 300ms 平滑过渡

📝 标题效果

  • group-hover:text-purple-600 - 悬停时文字变色
  • transition-colors - 颜色平滑过渡

🔘 按钮效果

  • opacity-0 → group-hover:opacity-100 - 淡入
  • translate-y-2 → group-hover:translate-y-0 - 向上移动

🏷️ 角标效果

  • -right-12 → group-hover:right-4 - 从右侧滑入
  • absolute - 绝对定位
