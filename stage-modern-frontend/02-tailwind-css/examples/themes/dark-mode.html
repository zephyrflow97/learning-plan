代码示例

<!-- 暗黑模式用法 -->
<div class="bg-white dark:bg-gray-900">
  <h1 class="text-gray-900 dark:text-gray-100">标题</h1>
  <p class="text-gray-600 dark:text-gray-400">段落</p>
</div>

<!-- 配置 Tailwind -->
tailwind.config = {
  darkMode: 'class', // 或 'media' 跟随系统
}

实现原理

当 <html> 标签有 dark 类名时,所有 dark: 前缀的样式生效。
